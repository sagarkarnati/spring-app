# Spring Swagger implementation with springfox

------------------------------------------------------------

**you can access swagger by going to ULR **http://localhost:8080/spring-swagger-springfox/swagger-ui.html#/ 

This is the basic Swagger setup for Spring REST services,we do a lot more customizations.
you can get more information from the link : http://springfox.github.io/springfox/docs/current/
